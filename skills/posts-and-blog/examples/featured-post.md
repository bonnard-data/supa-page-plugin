Today we're launching supa.page — a hosted marketing-site product whose
editor is **Claude Code in your terminal**.

## Why this exists

Every other marketing-site tool starts from "we have a browser canvas
and a drag-and-drop UI". That's fine if you're a designer mid-flow. It's
hell if you're a founder who wants to ship a landing page between meetings
and never touch a UI again.

supa.page inverts the loop: the editor is the agent in your terminal.
JSON for structure, Markdown for prose, plugin commands for everything
else. No canvas, no chat UI, no log-in-and-poke. Edit, sync, publish.
